# mactools

macOS 日常效率工具集，通过 Raycast 快捷调用。

## 文件操作 (file/)

| 脚本 | 功能 |
|------|------|
| `downloads_organizer.py` | Downloads 目录按扩展名自动分类整理 |
| `smart_rename.py` | AI 驱动的文件智能重命名（分析、分组、批量重命名） |
| `file_copy.py` | 复制 Finder 选中的文件 |
| `file_print.py` | 打印 Finder 选中的文件 |
| `folder_paste.sh` | 将粘贴板中的文件粘贴到 Finder 当前目录 |

## 系统工具 (system/)

| 脚本 | 功能 |
|------|------|
| `sys_app_launcher.py` | 应用启动器 |
| `display_1080.sh` | 切换显示器到 1080p 分辨率 |
| `display_4k.sh` | 切换显示器到 4K 分辨率 |
| `dingtalk_gov.sh` | 启动政务钉钉 |
| `create_reminder.sh` | 创建 Apple 提醒事项 |

## 窗口管理 (window/)

| 脚本 | 功能 |
|------|------|
| `yabai.py` | Yabai 窗口管理统一工具（浮动、跟随鼠标、布局切换等） |

## 公共库 (lib/)

clipboard, display, env, file_ops, finder, usage_log, llm_client, common.sh
